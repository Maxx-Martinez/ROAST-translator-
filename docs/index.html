3D Cartesian is recommended. It uses straight-line Euclidean distances between 3D scalp coordinates, not distance measured along the curved scalp surface.
Methodology note

3D Cartesian uses x, y, z coordinates. Movement and pairwise distance are straight chord distances through space. This version does not yet calculate true geodesic distance along a scalp mesh. 2D is primarily useful for visualization, debugging, and comparison with the cap map.
